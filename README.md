# RightGravity

피하기 게임으로 전환된 2D 종스크롤 게임

## 조작법

- **방향키**: 기체 이동
- **X**: 폭탄 (화면 전체 공격 및 적 제거)
- **Shift**: 저속 이동 (정밀 회피)

## 게임 특징

- **피하기 게임**: 적 비행기를 피하며 생존하는 게임
- **폭탄**: 화면 전체 공격 및 적 제거, 일시적 무적 (X키)
- **적 스폰**: 0.4초마다 적 비행기가 등장
- **하이스코어**: 로컬 스토리지에 최고 기록 저장

## 실행 방법

1. 웹 브라우저에서 `index.html` 파일을 열기
2. 또는 로컬 서버 실행:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (http-server)
   npx http-server
   ```
3. 브라우저에서 `http://localhost:8000` 접속

## 기술 스택

- Phaser.js 3.70.0
- HTML5 Canvas
- JavaScript (ES6+)

## 게임 플레이 팁

1. **저속 이동 활용**: Shift 키를 누르면 정밀한 회피가 가능합니다
2. **폭탄 전략**: 위험한 상황에서 폭탄을 사용하면 모든 적을 제거하고 무적 상태가 됩니다
3. **생존**: 적 비행기에 부딪히지 않도록 조심하세요!
